Weather App

Weather App - Client Application
A Simple AngularJS single page application that connects to the Backend API.

  1. Allow a user to enter a zip code.
  2. Display a three day forecast for the entered zip code.
  3. Display a spinning circle over the forecast while waiting for API data to load to give the user some indication that something is happening while you wait for data.

Weather App - Backend
A RESTful API built using Node and Express 4

  1. Receive API request from Client Application for zip code. Example: 60203.
  2. Check stored cache implemented in node-cache module for data on 60203 that is less than 1 hour old.
  3. If data is not stale, return's it to the user making the API request.
  4. if data is stale (older than 60 minutes) or non-existent, goes to the wunderground API to get the data, update cache and return's the data.
